== Description ==

uListing Compare - Listing Compare WordPress Plugin
uListing Compare Plugin is perfect for a listing management system on your WordPress site. It is fully responsive, highly customizable and works with nearly any WordPress theme.

Especially, uListing makes it simple to manage real estate listings, as well as car listings and general classified ads.

### uListing Plugin Key Features:
* Listing

== Installation ==
This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Please find more details on Plugin Installation in documentation
4. Set Up Page in Menu -> Listings.
